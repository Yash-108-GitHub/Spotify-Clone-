## Spotify UI Clone
Click here to view the project:

Live Demo: https://spotify-clone-zly9.onrender.com

This project is a static replica of Spotify's music player interface, built using only HTML and CSS. It recreates the layout and design of a modern music streaming platform with a focus on responsive structure and clean UI.

### Tech Stack
- HTML5
- CSS3 (Flexbox, Grid)

### Features
- Left sidebar with navigation (Home, Search, Your Library)
- Top navbar with profile and settings section
- Main content area displaying playlists, albums, and songs
- Sticky bottom music player with controls (play/pause, skip, volume)
- Responsive layout using Flexbox and CSS positioning

### Highlights
- Designed a music streaming UI clone using pure HTML and CSS
- Implemented structured layout using Flexbox and CSS positioning
- Focused on pixel-perfect design and modern UI layout techniques

### Note
- This is a frontend-only project (no backend or music functionality)
- Built for learning and practicing UI/UX design concepts
